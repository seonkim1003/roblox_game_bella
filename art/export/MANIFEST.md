# Runtime Export Manifest

Reviewed: 2026-07-13

There are currently no binary art exports. Premiere sets are generated from reviewed, checked-in Luau source at runtime, so there are no external asset IDs or placeholder exports to track.

Release review requires:

- a successful Rojo build;
- camera-marker presence for every authored set and transition;
- world-budget audit attributes on the generated world root;
- a visual pass in Studio at phone, tablet, and desktop aspect ratios.
